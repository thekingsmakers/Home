My Work & Projects

Welcome to my portfolio. Below are some of the custom scripts and unique solutions I've created to streamline IT operations, automate administrative tasks, and enhance endpoint security.

  • Password Expiration Notification Script - A GUI-based PowerShell tool that alerts users when their passwords are nearing expiration and provides an intuitive way to update passwords.
  • BitLocker Compliance Checker - A script that evaluates device compatibility for BitLocker silent encryption by analyzing PCR7 Configuration and Kernel DMA Protection.
  • Wallpaper Deployment Tool - Deploys custom wallpapers across devices through Intune, ensuring branding consistency across the enterprise.
  • Camera Restriction by Geolocation - A PowerShell script that disables or enables device cameras based on geographic location.
  • Trust Relationship Fixer - A script deployed via Intune that restores domain trust relationships, preventing costly reimaging or device recovery operations.
  • Conditional Access Policies - I have implemented and enforced numerous Conditional Access Policies to secure enterprise data, ensuring compliance with MFA, device health checks, and geofencing measures.
  • Intune Application Deployment - Experience deploying and managing Win32 and Line-of-Business apps, leveraging Intune for seamless software distribution across diverse endpoints.
  • SCCM Patch Management - Created automation scripts for patch deployment, compliance reporting, and remediation of failed deployments, reducing system vulnerabilities across the network.
  • Endpoint Security Configurations - Developed PowerShell scripts to configure Defender ATP policies, restrict USB usage, and manage firewall settings across endpoints.

For more of my scripts and solutions, please follow me on GitHub and Twitter.
